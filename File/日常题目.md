## 🎆简介
基础题的目的不是考察各位同学的能力😉，而是为了让各位同学快速入门这个专业，保证后续学习的效率🎠。
**所以本部分的题目不计入考察内容，也不需要提交😮，只要自己搞懂题目考察的内容即可。**


💡在本部分，你将接触到以下内容：

- 计算机基础知识
- markdown 语法
- git & github
- linux 系统
- 编程语言的入门知识
- IDE
- python入门
- 一些推荐书籍
